UnoCss配置 ​

UnoCss 样式查询表地址

https://unocss.dev/interactive/

UnoCSS 是即时原子 CSS 引擎，旨在灵活和可扩展。核心是 un-opinionated，所有的 CSS 实用程序都是通过 JavaScript 提供的。

1.安装配置 ​

shell
npm install -D unocss
1
js
// vite.config.ts
import UnoCSS from 'unocss/vite';
import { defineConfig } from 'vite';

export default defineConfig({
    plugins: [UnoCSS()]
});

2.创建一个uno.config.ts文件 ​

其中 presetRemToPx 需要单独安装配置 npm install -D @unocss/preset-rem-to-px

UnoCss 默认是 rem，开发的项目需使用 scale 来匹配不同分辨率，需要全部 px 为单位，这个插件可以使默认 rem 单位转为 px 单位

https://unocss.dev/presets/rem-to-px

注：有时刚修改的规则或快捷方式在使用是不会立即生效

使用 ctrl+shift+p 输入 unocss，选择 Reload UnoCss

js
// uno.config.ts
import {
    defineConfig,
    // 属性化模式，属性冲突时，可以通过默认un-前缀来解决：<div m-2 rounded text-teal-400 >代替class</div>
    presetAttributify,
    // 可以自定义图标转换
    presetIcons,
    presetTypography,
    // 预设，可以自定义
    presetUno,
    // 使用你自己的函数来获取字体来源
    presetWebFonts,
    // 别名，默认启用,指令：@apply等
    transformerDirectives,
    /**
     * 变型组Transformer，方便书写
     * 前缀组合：class="hover:bg-gray-400 hover:font-medium font-light font-mono"
     * 将转变为：class="hover:bg-gray-400 hover:font-medium font-light font-mono"
     */
    transformerVariantGroup
} from 'unocss';
import presetRemToPx from '@unocss/preset-rem-to-px';

export default defineConfig({
    rules: [
        // [/^m-(\d+)$/, (match) => ({ margin: `${+match[1] * 10}px` })]
    ],
    shortcuts: [
        [
            /^flex(-1|-col|-wrap)?\[([a-z]*)(?:,([a-z]*))?\]$/,
            (match) =>
                `flex flex${match[1] ? match[1] : ''} items-${match[2]} ${
                    match[3] != undefined ? 'justify-' + match[3] : 'justify-' + match[2]
                }`
        ]
        // 其他更多快捷方式在下面表格
    ],
    theme: {
        colors: {
            // ...
        }
    },
    presets: [
        presetUno(),
        presetAttributify(),
        presetIcons(),
        presetTypography(),
        presetWebFonts({
            fonts: {
                // ...
            }
        }),
        presetRemToPx({ baseFontSize: 4 }) // rem转px
    ],
    transformers: [transformerDirectives(), transformerVariantGroup()]
});
